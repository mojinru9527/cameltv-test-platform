## 来源 1: namethatui.com (web)

### 交互组件 (Component Gallery)
| 组件 | 说明 | 测试平台适用场景 |
|------|------|-----------------|
| Progress Ring | 环形进度指示器 | 测试执行进度、通过率仪表盘 |
| Spinner | 旋转加载动画 | 页面加载、数据请求中 |
| Progress Bar | 线性进度条 | 任务执行进度、批量操作进度 |
| Text Scramble (Decode Effect) | 字符解码动画效果 | 数据刷新、状态变化时的文字动画 |
| Skeleton | 骨架屏加载占位 | 列表/表格/卡片加载态 |
| Backdrop/Overlay | 背景幕布/覆盖层 | 弹窗/抽屉/全屏加载遮罩 |
| Easing (Timing Functions) | 缓动函数可视化 | 全局动画节奏定义 |

### 设计风格
| 风格 | 说明 |
|------|------|
| 液态玻璃 (Liquid Glass) | 毛玻璃效果，半透明背景 + backdrop-filter + 边框高光 |

## 来源 2: learnui.qiaomu.ai (web)

### 交互组件
| 组件 | 说明 | 测试平台适用场景 |
|------|------|-----------------|
| 轻提示 (Snackbar/Toast) | 操作结果反馈浮层 | 保存成功、删除确认、导入完成 |
| 标签页 (Tabs) | 内容区域切换 | 详情面板多视图、设置页分类 |

### 设计风格
| 风格 | 核心特征 |
|------|---------|
| 液态玻璃 (Liquid Glass) | 半透明、毛玻璃、光晕、大圆角 |
| 赛博朋克 (Cyberpunk) | 深色底、霓虹色(青/品红/黄)、终端字体、发光边框 |
| 黏土拟态 (Claymorphism) | 柔和3D、内阴影、大圆角、粉彩色(紫/桃/蓝)、膨胀感 |
| 苹果 (Apple) | 极简白色、大留白、细字体、微妙阴影、SF式圆角 |
| xAI | 深色科技感、纯净排版、AI产品气质、grok风格 |
| ClickHouse | 数据密集、暗色主题、列式数据表、技术型排版 |

## 与现有主题的差异化分析

现有 5 个主题: `blue` / `dark-minimal` / `warm` / `nature` / `liquid`
- `liquid` 已覆盖液态玻璃风格 → 新主题不再重复此方向
- `blue` 是通用企业风 → 新主题需要更极端的差异化
- 缺位风格: **赛博朋克**、**黏土拟态**、**苹果极简**、**AI实验室风**
