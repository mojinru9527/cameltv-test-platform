> 日期：2026-07-20 | 产品部门 | 对应核查报告 41 项功能声明 → 12 项落地

## 问题陈述

2026-07-20 对最新 develop (`a5f868d`) 代码核查发现：用户声明的 41 项「已完成」功能中，仅 12 项（29%）真正落地。三个模块存在显著差距：

1. **后端用例服务**：软删除全链路（Model → Service → API）从未实现，用例仍是物理删除；搜索仅覆盖 3 个字段而非声明的 7 个
2. **前端用例列表**：9 项界面改动（列重排、新列、隐藏旧列、去导出按钮、搜索扩展）全部未执行
3. **接口测试模块**：DebugTab 接线断裂（prop 存在但不传入），默认 Tab 未切换，搜索未扩展

## 为什么用户关心

- **数据安全**：物理删除用例不可恢复，操作失误导致测试资产永久丢失
- **操作效率**：搜索只能搜标题/编号/API 三个字段，无法搜步骤内容或预期结果，查找效率低
- **信息密度**：列表看不到前置条件/步骤/预期结果，必须点进详情才能审阅
- **调试体验**：从接口资产点击「调试」按钮，跳转到 DebugTab 后参数全是空的——预填功能写了但没接线

## 用户故事

### US-1：删除用例可恢复

**As** 测试工程师  
**I want** 删除用例时标记为「已删除」而非从数据库抹掉  
**So that** 误删的用例可以被恢复，审计时可追溯删除历史  

**验收标准**：
- Given 用例列表中有 10 条用例，When 删除其中 1 条，Then 该用例 `is_deleted=true`，`list_cases` 不再返回它
- Given 已删除 3 条用例，When 删除其所属模块，Then 该模块下 3 条用例全部 `is_deleted=true`
- Given 批量选择 5 条用例，When 执行批量删除，Then 5 条全部 `is_deleted=true`，0 条物理删除

### US-2：搜索覆盖更多字段

**As** 测试工程师  
**I want** 在搜索框输入「登录失败」时能匹配到步骤或预期结果中包含该词条的用例  
**So that** 我不需要记住用例标题或编号就能找到相关用例  

**验收标准**：
- Given 用例 A 标题「用户登录」、用例 B 步骤中含「登录失败提示」，When 搜索「登录失败」，Then 返回 B（不要求返回 A）
- Given 搜索框为空，When 直接回车，Then 返回全部用例

### US-3：列表直接展示关键信息

**As** 测试评审者  
**I want** 在列表视图直接看到前置条件、操作步骤、预期结果  
**So that** 我不需要逐个点开 900+ 条用例就能快速浏览和比对用例内容  

**验收标准**：
- Given 用例列表页，When 打开页面，Then 表格包含「模块名称」「用例标题」「用例等级」三列且等级为独立 Badge 列
- Given 用例列表页，When 打开页面，Then 表格包含「前置条件」「操作步骤」「预期结果」三列（单行省略）
- Given 用例列表页，When 打开页面，Then 不显示「API」列和 Excel/Xmind 导出按钮
- Given 域/模块/优先级下拉，When 点击下拉，Then 包含显式「全部」选项

### US-4：接口资产点击调试自动填参数

**As** 接口测试人员  
**I want** 从接口资产列表点击某个接口的「调试」按钮后，DebugTab 自动填入该接口的路径、Query、Header、Body  
**So that** 我不需要手动复制粘贴参数就能快速发起调试请求  

**验收标准**：
- Given 接口资产页有一个 `GET /api/user/info` 接口，When 点击「调试」按钮，Then 自动切换到快速调试 Tab，Method=GET，路径已填入
- Given 接口资产有一个带 Query 参数的接口，When 点击「调试」，Then Query 参数表已预填
- Given 接口资产页，When 打开页面，Then 默认显示「接口资产」Tab（非「快速调试」）

### US-5：新建用例表单验证加强

**As** 测试工程师  
**I want** 新建用例时模块、步骤、预期结果为必填  
**So that** 我不会意外提交不完整的用例  

**验收标准**：
- Given 新建用例弹窗，When 模块为空并点击保存，Then 显示「请选择模块」校验错误
- Given 新建用例弹窗，When 步骤为空并点击保存，Then 显示「请填写操作步骤」校验错误
- Given 新建用例弹窗，When 预期结果为空并点击保存，Then 显示「请填写预期结果」校验错误
- Given 新建用例弹窗，When 打开，Then 不显示「关联引用」字段
