# CamelTv 基线需求（用户端 14.0.0 之前存量）

>       完整逐版本更新日志见 [更新日志-用户端原型-完整版](../../../产品需求/更新日志-用户端原型-完整版.md)
> 来源: 蓝湖原型(用户端+运营后台) + [用户端原型-需求分析](用户端原型-需求分析.md) + [运营后台-需求分析](运营后台-需求分析.md)
> 状态: 基线存档
