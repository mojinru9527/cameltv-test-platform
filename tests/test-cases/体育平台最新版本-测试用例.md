# 体育预测平台最新版本 — 测试用例

> 依据 `tests/test-case-standards/` 规范产出。用例字段、级别定义见 `功能测试用例规范.md`;接口检查点见 `API接口测试方案.md`。

## 一、文档简介

- **测试目的**:验证本版本「首单不中退币活动、充值额外赠送骆驼币、预测/UGC 列表、UGC 详情、骆驼币充值、首页预测推荐」六大模块功能与需求一致,保证关键路径稳定、边界与异常可控。
- **测试范围**:上述六大模块的功能、界面、接口、相关异常与并发场景。
- **ID 规则**:功能用例 `TC-[模块]-序号`;接口用例 `API-TC-[模块]-序号`。
- **级别**:P0=关键路径/基本流;P1=异常流/边界值;P2=GUI/体验。
