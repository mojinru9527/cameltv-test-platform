## 2. OpenCV 安装

### 2.1 卸载旧版本（如之前装过）

```cmd
pip uninstall opencv-python
pip uninstall opencv-contrib-python
```

执行结果：`[________________]`

### 2.2 下载正确版本

| 步骤 | 操作 | 填写/确认 |
|------|------|-----------|
| 1 | 访问下载页面 | https://www.lfd.uci.edu/~gohlke/pythonlibs/#opencv |
| 2 | 下载文件 | `opencv_python-4.0.1-cp37-cp37m-win_amd64.whl` |
| 3 | 文件存放路径 | `[________________]` (例如: C:\Users\[用户名]\Downloads\) |

### 2.3 安装

打开cmd，逐条执行：

```cmd
python -m pip install --upgrade pip

cd [你的下载文件夹路径，例如 C:\Users\你的用户名\Downloads]

pip install opencv_python-4.0.1-cp37-cp37m-win_amd64.whl

pip install opencv-contrib-python
```

每步执行结果：

| 步骤 | 执行结果（填入） |
|------|-----------------|
| pip升级 | `[________________]` |

| contrib安装 | `[________________]` |

### 2.4 安装依赖库

```cmd
pip install matplotlib

```

执行结果：`[________________]`

### 2.5 解决DLL问题（如果cv2导入失败）

如果导入cv2时提示DLL错误，按以下步骤操作：

| 步骤 | 操作 | 完成 |
|------|------|------|
| 1 | 找到文件: `C:\Users\[用户名]\AppData\Local\Programs\Python\Python37\Lib\site-packages\cv2\` | ☐ |
| 2 | 复制 `cv2.cp37-win_amd64.pyd` 文件 | ☐ |
| 3 | 粘贴到: `C:\Users\[用户名]\AppData\Local\Programs\Python\Python37\Lib\` | ☐ |

## 3. Tesseract-OCR 安装

### 3.1 下载安装

| 步骤 | 操作 | 填写/确认 |
|------|------|-----------|
| 1 | 访问下载页面 | https://github.com/UB-Mannheim/tesseract/wiki |
| 2 | 下载文件 | `tesseract-ocr-w64-setup-v5.0.0.20190623.exe` |
| 3 | 安装路径 | `C:\Program Files\Tesseract-OCR\` |

### 3.2 环境变量配置

| 步骤 | 操作 | 完成 |
|------|------|------|
| 1 | 右键"此电脑" → 属性 → 高级系统设置 → 环境变量 | ☐ |
| 2 | 在"系统变量"中找到 `Path`，点击编辑 | ☐ |
| 3 | 新建 → 添加 `C:\Program Files\Tesseract-OCR\` | ☐ |
| 4 | 新建系统变量 → 变量名: `TESSDATA_PREFIX` | ☐ |
| 5 | 变量值: `C:\Program Files\Tesseract-OCR\tessdata` | ☐ |
| 6 | 点击"确定"保存所有设置 | ☐ |

### 3.3 验证Tesseract安装

```cmd
tesseract -v
```
