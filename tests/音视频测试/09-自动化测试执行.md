# 09-自动化测试执行

> **模块说明**: 本模块提供自动化批处理脚本，将测试执行流程（创建目录→记录环境→处理视频→分析延迟→输出结果）串联起来，减少手动操作错误，提高重复测试效率。
> **前置条件**: 已完成 [01-环境安装指南](01-环境安装指南.md) 和 [02-测试素材准备](02-测试素材准备.md)
> **适用场景**: 需要反复执行相同测试流程时（如每日回归测试、多场景批量测试）

---

## 2. 一键测试脚本：run_test.bat

### 2.1 创建脚本

在 `C:\直播测试脚本\` 下创建 `run_test.bat`，复制以下内容：

```batch
@echo off
chcp 65001 >nul
setlocal enabledelayedexpansion

echo ==========================================
echo     直播延迟测试自动化脚本
echo ==========================================
echo.

REM ============ [可修改配置区域] ============
set SCRIPTS_DIR=C:\直播测试脚本
set RESULTS_BASE=C:\直播测试结果
set PYTHON_EXE=python
REM =========================================

REM 1. 获取场景名称
set TEST_SCENARIO=%1
if "%TEST_SCENARIO%"=="" (
    set /p TEST_SCENARIO="请输入测试场景名称 (如: normal/wifi_weak/4g): "
)

REM 2. 生成时间戳目录名
set TIMESTAMP=%date:~0,4%%date:~5,2%%date:~8,2%_%time:~0,2%%time:~3,2%%time:~6,2%
set TIMESTAMP=%TIMESTAMP: =0%
set TEST_DIR=%RESULTS_BASE%\test_%TIMESTAMP%_%TEST_SCENARIO%

echo [1/6] 创建测试目录: %TEST_DIR%
mkdir "%TEST_DIR%"
cd /d "%TEST_DIR%"

REM 3. 记录测试环境信息
echo [2/6] 记录测试环境信息...
echo ===== 测试环境信息 ===== > test_info.txt
echo 测试场景: %TEST_SCENARIO% >> test_info.txt
echo 开始时间: %date% %time% >> test_info.txt
echo 测试脚本版本: 1.0 >> test_info.txt
echo. >> test_info.txt

REM 获取网络信息
echo --- 网络信息 --- >> test_info.txt
ipconfig | findstr /i "IPv4" >> test_info.txt
echo. >> test_info.txt

REM 获取系统信息
echo --- 系统信息 --- >> test_info.txt
echo 计算机名: %COMPUTERNAME% >> test_info.txt
echo 用户名: %USERNAME% >> test_info.txt
echo.

REM 4. 等待手动录制
echo [3/6] 请手动完成以下操作:
echo   (1) 打开OBS开始推流
echo   (2) 打开Bandicam开始录制
echo   (3) 录制完成后停止Bandicam
echo.
set /p RECORDING_FILE="请输入录制文件完整路径 (可直接拖拽文件到此处): "

REM 去掉路径中可能的引号
set RECORDING_FILE=%RECORDING_FILE:"=%

if not exist "%RECORDING_FILE%" (
    echo 错误: 文件 "%RECORDING_FILE%" 不存在！
    pause
    exit /b 1
)

REM 5. 复制录制文件到测试目录
echo.
echo [4/6] 复制录制文件...
copy "%RECORDING_FILE%" "%TEST_DIR%\recording.mp4" >nul
echo 已复制到: %TEST_DIR%\recording.mp4

REM 6. 运行OCR识别
echo.
echo [5/6] 运行视频时间戳识别...
%PYTHON_EXE% "%SCRIPTS_DIR%\opencv.py" "%TEST_DIR%\recording.mp4"

if %ERRORLEVEL% NEQ 0 (
    echo 错误: opencv.py 执行失败！
    pause
    exit /b 1
)

REM 7. 运行延迟分析
echo.
echo [6/6] 运行延迟分析...
%PYTHON_EXE% "%SCRIPTS_DIR%\analysis.py" "%TEST_DIR%\recording.log"

if %ERRORLEVEL% NEQ 0 (
    echo 错误: analysis.py 执行失败！
    pause
    exit /b 1
)

REM 8. 整理结果
echo.
echo 整理结果文件...
if exist "%TEST_DIR%\recording_chart.png" (
    echo 图表: recording_chart.png
)
if exist "%TEST_DIR%\recording_results.txt" (
    echo 数据: recording_results.txt
)

REM 9. 完成
echo.
echo ==========================================
echo     测试完成！
echo     结果目录: %TEST_DIR%
echo ==========================================

REM 打开结果目录
explorer "%TEST_DIR%"

pause
```

### 2.2 使用方法

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 准备好录制文件 | 按照对应测试模块的步骤完成录制 |
| 2 | 双击 `run_test.bat` | 或在cmd中运行 |
| 3 | 输入场景名称 | 例如: `normal`, `weak_wifi`, `4g_test` |
| 4 | 按提示拖入录制文件 | 直接把mp4文件拖到cmd窗口即可 |
| 5 | 等待自动处理完成 | 处理时间取决于视频长度 |
| 6 | 查看结果 | 自动弹出结果文件夹，包含chart.png和results.txt |

当需要一次性测试多个场景时使用。

在 `C:\直播测试脚本\` 下创建 `batch_test.bat`：

```batch
@echo off
chcp 65001 >nul

echo ==========================================
echo     批量直播测试脚本
echo ==========================================
echo.
echo 请确保:
echo   (1) OBS已配置好推流
echo   (2) Bandicam已准备就绪
echo   (3) 测试手机APP已登录
echo.
echo 以下场景将依次执行:
echo   场景1: 正常网络
echo   场景2: WiFi 2.4GHz
echo   场景3: 4G移动网络
echo.

pause

REM ===== 场景1 =====
echo.
echo ===== 场景1: 正常网络 =====
echo 请切换网络到正常WiFi，然后开始录制...
echo 录制完成后按任意键继续...
pause >nul
call run_test.bat normal

REM ===== 场景2 =====
echo.
echo ===== 场景2: WiFi 2.4GHz =====
echo 请切换网络到2.4GHz WiFi，然后开始录制...
echo 录制完成后按任意键继续...
pause >nul
call run_test.bat wifi_24g

REM ===== 场景3 =====
echo.
echo ===== 场景3: 4G移动网络 =====
echo 请关闭WiFi，使用移动数据，然后开始录制...
echo 录制完成后按任意键继续...
pause >nul
call run_test.bat mobile_4g

echo.
echo ==========================================
echo     所有场景测试完成！
echo     结果保存在: C:\直播测试结果\
echo ==========================================
pause
```

## 4. 结果整理脚本：collect_results.bat

将所有测试结果汇总到一个Excel兼容的CSV文件。

在 `C:\直播测试脚本\` 下创建 `collect_results.py`：

```python
import os
import glob
import csv
from datetime import datetime

def collect_results():
    results_dir = "C:\\直播测试结果"
    output_file = os.path.join(results_dir, f"汇总报告_{datetime.now().strftime('%Y%m%d_%H%M%S')}.csv")
    
    # 收集所有results.txt文件
    result_files = glob.glob(os.path.join(results_dir, "**", "*_results.txt"), recursive=True)
    
    if not result_files:
        print("未找到任何测试结果文件！")
        return
    
    print(f"找到 {len(result_files)} 个测试结果文件")
    
    all_results = []
    
    for rf in result_files:
        test_dir = os.path.dirname(rf)
        test_name = os.path.basename(test_dir)
        
        with open(rf, 'r', encoding='utf-8') as f:
            content = f.read()
        
        # 解析结果
        result = {'测试名称': test_name, '结果文件': rf}
        for line in content.split('\n'):
            if ':' in line:
                key, value = line.split(':', 1)
                result[key.strip()] = value.strip()
        
        all_results.append(result)
    
    # 写入CSV
    if all_results:
        fieldnames = ['测试名称'] + [k for k in all_results[0].keys() if k != '测试名称']
        
        with open(output_file, 'w', newline='', encoding='utf-8-sig') as f:
            writer = csv.DictWriter(f, fieldnames=fieldnames[:20])  # 限制列数
            writer.writeheader()
            for r in all_results:
                writer.writerow(r)
        
        print(f"汇总报告已保存: {output_file}")
    else:
        print("无有效结果可汇总")

if __name__ == "__main__":
    collect_results()
```

**使用方法**：

```cmd
python C:\直播测试脚本\collect_results.py
```

## 7. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 双击bat文件闪退 | 在文件末尾已有`pause`，如果还闪退，在cmd中手动运行bat查看错误 |
| 中文显示乱码 | bat文件开头已有`chcp 65001`，如果还乱码，检查cmd字体是否支持UTF-8 |
| "python不是内部命令" | Python未添加到PATH，回到[01-环境安装指南](01-环境安装指南.md)检查 |
| 拖入文件路径包含空格 | 脚本已处理引号，如果还报错，手动输入路径并加双引号 |
| 批处理找不到文件 | 确认所有脚本都在`C:\直播测试脚本\`下，路径不要有中文 |
