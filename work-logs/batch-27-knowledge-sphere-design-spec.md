shadcn/ui + Radix + Tailwind + CVA；Token 走语义类（bg-muted / text-muted-foreground / border / variant）。图谱使用 vis-network（已有集成）。

**交互**：
- 左侧版本列表：点击切换版本，当前版本高亮。支持搜索/过滤
- 三列平台卡片：每列独立滚动，模块卡片可折叠
- 模块卡片：header 显示模块名+页面数，body 显示页面列表（每项可点击跳转到蓝湖原型页）
- 运营后台区域：默认折叠，点击展开。每项显示关联箭头指向用户端模块
- 说明附件：文件图标+文件名，点击下载/预览
- **关联编辑**：hover 运营后台模块时出现「关联」按钮，可拖拽到目标用户端模块建立关联
- 🆕 v1.1: 模块卡片 hover 展示缩略图轮播（screenshot_urls）
- 🆕 v1.2: 页面项右侧显示跳转关系图标（outgoing/incoming 数量标记）
- 🆕 v1.3: 模块含 dynamic_filter 交互时显示紫色标记；底部显示全局导航条

**vis-network 层级布局配置**：
```js
// 不使用 forceAtlas2Based 物理引擎，改用 hierarchical layout
{
  layout: {
    hierarchical: {
      enabled: true,
      direction: 'UD',        // 上→下
      sortMethod: 'directed',
      levelSeparation: 120,   // 层级间距
      nodeSpacing: 80,        // 同级节点间距
    },
  },
  physics: false,              // 层级模式下禁用物理引擎
}
```

**交互**：
- 双击节点：展开/折叠其子节点
- 单击节点：右侧面板显示详情
- 拖拽：仅限同一层级内排序
- 右键菜单：编辑/删除/查看关联/跳转蓝湖
- 缩放：滚轮 + 工具栏按钮
